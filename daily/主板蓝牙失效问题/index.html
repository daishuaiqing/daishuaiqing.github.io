技嘉B550I AORUS PRO AX 主板蓝牙失效问题

2021年07月08日
0 条评论

去年组装了一台台式机电脑，算是个中配，清单后续补上。电脑主要配件都是在京东购买的，出了问题退换比较方便，其他配件全部在淘宝采购，京东也是可以砍价的，就是只能砍很少一部分，淘宝联系店家可以优惠很多，有不少配件是薅的羊毛。

两次搬家，都是拖着它到处跑，上次搬家拿出来使用的时候发现出了点小问题，蓝牙的图标不见了，蓝牙与wifi在同一个部件上，一般很难坏掉。网上看了一下，大部分人都是说驱动没有安装好，我就在官网下载了一些驱动重新装了一下，但还是不行，心烦气躁之下把系统重新装了一遍，结果还是不行。

最后实在没办法，找技嘉官方客服问了一下，客服也很尽责，把主板跟系统同版本号的组了一套测试，证明了主板的蓝牙与系统确实没有不兼容，他们说要给我换个主板，我想了下还是算了，太麻烦了，蓝牙我也不常用，有问题也不是大问题，主要还是小机箱拆主板太麻烦，还有就是换一次要一周时间，等不起啊。

也没几天时间，在贴吧上看到有人解决了这个问题，说usb插线太多，电流有干扰，导致蓝牙失灵，索性试了下，发现不可以。

下面看到还有人通告拔电源解决了这个问题，我想了下也有道理，毕竟插着电源主板是一直带电的，断下点可能就恢复了。拔掉电源等了一会儿又插上，发现蓝牙还真的出来了，看来这个主板确实有点小Bug存在的，索性也给官方提了一下，不知道升级固件能不能解决。强迫症患者表示，尽管蓝牙用不到，它是没问题的还是舒服的。

总结下蓝牙无法使用的几个可能：

  1. 电脑可能没有蓝牙模块
  2. 电脑驱动程序不兼容（升级驱动或者驱动降版本）
  3. 电脑系统不兼容（可以尝试升级或者回退版本）
  4. 硬件问题，电流干扰导致（拔电源，静置放点）
